OA progression prediction with IMI-APPROACH dataset


IMI-APPROACH dataset includes 6 sources of data: Biochemical markers (B), Clinical assessments(C), Questionnaires (Q), MRI (M), Xray (X), Tomography(T). Machine learning models are trained using each data source as well as all data combined to predict nine categories of OA progression.
